## English

Title:

History Record Lens - AI History Search | One-Click Cleanup

Summary:

Find Chrome browsing history with AI, maintain website lists, and clean local history with one click.

Description:

History Record Lens helps you find, review, and clean local Chrome browsing history without leaving the browser side panel.

It is designed for two everyday workflows:

- recalling a page from fuzzy memory with an AI-assisted history search
- maintaining a website list for quick access and one-click cleanup

Core features:

- Search local Chrome browsing history with keyword and time-range filters
- Use an OpenAI-compatible model API to recover pages when you remember the idea but not the exact title
- Configure multiple model services and switch between them
- Maintain a website list for quick access
- Clean local Chrome history for selected websites with one click
- Enable automatic cleanup when matching websites are closed
- Switch the interface between Chinese and English, with browser-language default and local persistence

All website lists, settings, and browsing-history cleanup rules are stored locally in the browser. AI requests are sent only to the model service the user explicitly configures.
