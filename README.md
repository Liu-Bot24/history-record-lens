# 历史记录透镜

</div>

历史记录透镜是一款 Chrome 侧边栏扩展，用来查找、找回和清理本地浏览历史。它适合两类场景：想用大模型从模糊记忆里找回网页，或者想维护一份网站清单，对指定网站进行快捷访问和一键清理。

## 功能

- 搜索 Chrome 本地浏览历史，并按最近一小时、最近一天、最近七天或日期区间缩小范围。
- 使用 OpenAI 兼容接口的大模型进行语义找回，适合“记得大概意思，但想不起标题或关键词”的网页。
- 配置多个模型服务，填写 API 接口、API Key 和模型名称后即可切换使用。
- 维护网站清单，把常用网站加入快捷访问区。
- 对清单中的网站一键清理 Chrome 本地浏览历史。
- 开启自动清理后，在关闭清单中的网站时自动清理对应浏览记录。

## 安装

1. 在 GitHub Releases 下载最新版本的扩展压缩包。
2. 解压到本地一个固定目录。
3. 在 Chrome 中打开 `chrome://extensions/`。
4. 开启“开发者模式”。
5. 点击“加载已解压的扩展程序”，选择刚才解压出来的文件夹。

## 使用

### AI 查询历史记录

1. 在搜索框中输入关键词，可先做精确筛选。
2. 选择时间范围，或保持“不限”。
3. 在 AI 查询框里描述你记得的网页，例如“前几天看过一个工具或教程网页，想不起来名字了”。
4. 配置或选择模型服务，点击“AI 查询”。

AI 查询会把当前筛选范围内的历史记录发送到你配置的大模型服务。历史记录中存在敏感信息时，可能触发部分模型的内容安全审查而导致查询失败，可以清洗浏览记录或更换其他模型。

### 一键清理浏览记录

网站清单每行一个网站，格式如下：

```text
YouTube, https://www.youtube.com, youtube.com, youtu.be
bilibili, https://www.bilibili.com, bilibili.com
```

第一个逗号前是网站名称，第二段是快捷访问打开的网址，后面的域名用于匹配并清理浏览历史。也可以点击“添加当前网站”，把当前正在浏览的网站加入清单。

清单里的星标表示是否显示在快捷访问区，盾牌表示是否加入一键清理。清理只作用于 Chrome 本地浏览历史，不会清理 Cookie、缓存、Local Storage，也不会删除网站账号里的观看记录或搜索记录。

## 模型配置

| 配置项 | 说明 |
| --- | --- |
| API 接口 | OpenAI 兼容接口地址，例如 `https://api.openai.com/v1` |
| API Key | 你的模型服务密钥 |
| 模型名称 | 要调用的模型 ID |

API Key 保存在本机浏览器扩展存储中。请只在可信设备上使用，并确认你配置的 API 服务可信。

## 权限说明

- `history`：读取和删除 Chrome 本地浏览历史。
- `storage`：保存网站清单、模型配置和界面偏好。
- `tabs`：获取当前标签页，用于添加当前网站和关闭网站时自动清理。
- `sidePanel`：在 Chrome 侧边栏中打开扩展界面。
- `favicon`：显示网站图标。
- 可选网站访问权限：仅在测试或调用你配置的模型 API 时请求对应接口域名权限。
