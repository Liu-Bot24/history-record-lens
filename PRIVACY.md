# 隐私政策

历史记录透镜是一款用于查找、管理和清理 Chrome 本地浏览历史的浏览器扩展。

## 数据使用

扩展会在本机读取 Chrome 浏览历史，用于历史记录搜索、AI 语义查找、网站清单快捷访问、一键清理和自动清理功能。扩展会在本机保存用户配置的网站清单、清理设置、界面偏好和模型服务配置。

当用户主动配置模型服务并发起 AI 查询或连接测试时，扩展会向用户填写的模型 API 地址发送请求。AI 查询请求可能包含用于匹配的历史记录标题、网址、访问时间等信息。API Key 仅用于向用户配置的模型服务发起请求。

## 不会做的事

扩展不会把用户数据发送到开发者服务器，不会出售用户数据，不会将用户数据用于广告、信用评估、贷款或与扩展功能无关的用途。

扩展不会清理 Cookie、缓存、Local Storage，也不会删除网站账号内的观看记录、搜索记录或其他云端数据。

## 用户控制

用户可以自行选择是否配置模型服务、是否发起 AI 查询、是否启用自动清理，以及清理哪些网站的本地浏览历史。用户可以在扩展设置中删除或修改已保存的配置。
