# Privacy Policy

Last updated: `2026-05-21`

History Record Lens is a browser extension for finding, managing, and cleaning local Chrome browsing history.

## Information We Process

The extension may process the following information locally in your browser:

- Chrome browsing-history records such as page titles, URLs, visit times, visit counts, and typed counts
- website lists and cleanup rules created by the user
- model-service settings such as API endpoint, API key, and model name
- interface preferences such as language selection and sorting options

When you actively configure a model service and run an AI query or a connection test, the extension may send filtered browsing-history records to the model API endpoint you configured. These records can include titles, URLs, and visit times needed for semantic matching.

## How We Use Information

The information above is used only to provide the extension's core features, including:

- searching local browsing history
- AI-assisted recall of previously visited pages
- quick access to saved websites
- one-click cleanup of matching local history records
- automatic cleanup when matching websites are closed
- saving local settings and preferences

## Data Storage

- Website lists, cleanup rules, model-service settings, and interface preferences are stored locally in your browser.
- You can remove or change saved settings from the extension UI.
- When the extension is uninstalled, extension data is removed by the browser according to browser behavior.

## Network Requests

The extension does not send browsing-history data to developer-operated servers.

If you configure a model service, the extension may send requests to that model API endpoint when you:

- test the configured connection
- run an AI history query

These requests are sent only to the model service you choose, not to the developer.

## Data Sharing

- The extension does not sell user data.
- The extension does not use user data for advertising, credit evaluation, lending, or unrelated profiling.
- The extension does not sync your browsing-history cleanup lists to a cloud service.

## What The Extension Does Not Clean

The extension only cleans local Chrome browsing history entries.

It does not delete cookies, cache, Local Storage, watch history inside website accounts, search history inside website accounts, or other cloud-side records.

## User Control

You can decide whether to configure a model service, whether to run AI queries, whether to enable automatic cleanup, and which websites should be included in local history cleanup.
